| 交互 | Web UI | 支持参数调整、回测、结果展示 | ✅ |
| 工程 | 测试样例 | 配置与核心规则基础测试 | ✅ |

| 数据 | CSV缓存 | 支持行情与板块数据本地缓存 | ✅ |
